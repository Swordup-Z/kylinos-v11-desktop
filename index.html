What this series is for 这一系列项目要做什么

These projects collect practical work around KylinOS Desktop V11: system diagnosis, verified fixes, feature enhancement, desktop tools, UI verification, and AI desktop operations. 这一系列项目围绕麒麟桌面操作系统 V11 的实际使用场景展开：系统诊断、已验证修复、功能增强、桌面工具、界面验证，以及 AI 桌面操作能力。

Find system issues faster 更快定位系统问题

Skill repositories turn repeated repair experience into clear steps for AI tools: what to check, which logs matter, what risks to avoid, and how to verify a fix. skill 仓库把重复出现的修复经验整理成 AI 工具可直接使用的步骤：检查什么、看哪些日志、避开什么风险，以及如何验证修复结果。

Share verified improvements 沉淀已验证的增强能力

The planned enhancement skill will record safe, repeatable ways to improve the desktop system and document the required checks before applying changes. 计划中的增强 skill 会记录安全、可重复的桌面系统增强方法，并说明执行前需要检查的条件。

Build practical local apps 开发实用本地工具

Tool projects focus on daily desktop maintenance, disk usage analysis, Kaiming/KARE cleanup, UI verification, and stronger desktop control for AI tools. 工具项目面向日常桌面维护、空间占用分析、Kaiming/KARE 清理、界面验证，以及增强 AI 工具操作桌面的能力。

Open source repositories 已开源仓库

kylinos-v11-desktop-fix-skill

A reusable repair skill for AI-assisted diagnosis of KylinOS Desktop V11 issues. 用于 AI 辅助诊断麒麟桌面操作系统 V11 问题的可复用修复 skill。

It turns repeated repair work into clear routes, checklists, logs to inspect, risk notes, and validation steps. The goal is to help AI tools move from symptoms to verified fixes faster. 它把重复出现的修复工作整理成清晰的路由、检查清单、日志线索、风险说明和验证步骤，目标是帮助 AI 工具更快从现象定位到经过验证的修复方案。

Projects in development 正在开发的仓库

kylinos-v11-desktop-enhance-skill

A planned skill for verified enhancement workflows on KylinOS Desktop V11. 计划用于沉淀麒麟桌面操作系统 V11 已验证增强流程的 skill。

It will keep feature enhancement steps, pre-checks, rollback notes, and validation rules together, so AI tools can suggest changes with clearer safety boundaries. 它会把功能增强步骤、前置检查、回退说明和验证规则放在一起，让 AI 工具能在更清晰的安全边界内提出改动建议。

Planned 计划中

kylin-space-guard

A conservative desktop space cleaner for KylinOS Desktop V11. 面向麒麟桌面操作系统 V11 的保守型桌面空间清理工具。

It focuses on Kaiming/KARE and ostree disk usage, identifies old Kaiming app containers, and moves safe cleanup candidates into a rollback quarantine instead of deleting them directly. 它关注 Kaiming/KARE 和 ostree 空间占用，识别旧的 Kaiming 应用容器，并把安全清理候选移动到可回退隔离区，而不是直接删除。

Local development 本地开发中

kylinos-v11-desktop-computer-use

A lightweight X11/XTest helper for AI desktop observation and control. 用于增强 AI 桌面观察和操作能力的轻量 X11/XTest 辅助工具。

It can capture screens and windows, send real mouse and keyboard events, support UI verification, and make AI-driven desktop operations more reliable on KylinOS Desktop V11. 它可以截取屏幕和窗口，发送真实鼠标和键盘事件，支持界面验证，并让 AI 在麒麟 V11 桌面上的操作更可靠。
