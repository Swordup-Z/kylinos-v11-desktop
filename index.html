What this series is for 这一系列项目要做什么

These projects collect practical work around KylinOS Desktop V11: system diagnosis, verified fixes, feature enhancement, desktop tools, UI verification, and AI desktop operations. 这一系列项目围绕麒麟桌面操作系统 V11 的实际使用场景展开：系统诊断、已验证修复、功能增强、桌面工具、界面验证，以及 AI 桌面操作能力。

AI skills AI 技能

Find system issues faster 更快定位系统问题

Skill repositories turn repeated repair experience into clear steps for AI tools: what to check, which logs matter, what risks to avoid, and how to verify a fix. skill 仓库把重复出现的修复经验整理成 AI 工具可直接使用的步骤：检查什么、看哪些日志、避开什么风险，以及如何验证修复结果。

Enhancement 功能增强

Share verified improvements 沉淀已验证的增强能力

The planned enhancement skill will record safe, repeatable ways to improve the desktop system and document the required checks before applying changes. 计划中的增强 skill 会记录安全、可重复的桌面系统增强方法，并说明执行前需要检查的条件。

Desktop tools 桌面工具

Build practical local apps 开发实用本地工具

Tool projects focus on daily desktop maintenance, disk usage analysis, Kaiming/KARE cleanup, UI verification, and stronger desktop control for AI tools. 工具项目面向日常桌面维护、空间占用分析、Kaiming/KARE 清理、界面验证，以及增强 AI 工具操作桌面的能力。

A conservative desktop space cleaner for KylinOS Desktop V11. 面向麒麟桌面操作系统 V11 的保守型桌面空间清理工具。

  • Show Kaiming/KARE and ostree disk usage. 展示 Kaiming/KARE 和 ostree 空间占用。
  • Move old Kaiming app containers to a rollback quarantine. 把旧 Kaiming 应用容器移动到可回退隔离区。
