# Repository Rules

## Project Scope

- This repository is the static GitHub Pages project index for KylinOS Desktop V11 related work.
- Keep implementation limited to the static site files in this repository, mainly `index.html`, `styles.css`, `README.md`, and repository metadata.
- Do not place independent desktop tool implementation, build output, screenshots, or system repair knowledge in this repository.
- Use `$HOME/.os-fix-skill` for reusable KylinOS Desktop V11 system repair knowledge.
- Use `$HOME/.os-enhance-skill` for reusable KylinOS Desktop V11 feature enhancement knowledge.

## Website Changes

- Keep the first page as the usable project index, not a marketing landing page.
- Keep English and Chinese page text synchronized when changing visible copy.
- Keep project names exact and consistent with their repository names.
- The enhancement skill project name is `kylinos-v11-desktop-enhance-skill`; do not shorten it to `os-enhance-skill`.
- Preserve responsive layout behavior for both desktop and mobile viewports.

## Git Commits

- Use git author `Swordup-Z <swordup.zeng@gmail.com>` for commits.
- Do not include AI-related author, co-author, generator, or assistant attribution in commit messages.
- Avoid trailers and phrases such as `Co-authored-by: Claude`, `Co-authored-by: Codex`, `Generated with ...`, `AI-assisted`, or similar AI disclosure text.
- Keep commit messages focused on the actual repository change.
